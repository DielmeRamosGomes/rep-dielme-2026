1) Escreva um programa que defina um número secreto 
(por exemplo, 42). O programa deve pedir para o 
usuário tentar adivinhar o número. Enquanto o usuário 
errar, o programa deve dizer se o palpite foi "Muito 
alto" ou "Muito baixo" e pedir um novo palpite. 
Quando o usuário acertar, o programa exibe uma 
mensagem de parabéns e encerra.

2) Desenvolva um programa que peça ao usuário para 
digitar um número inteiro. O programa deve utilizar 
um laço para para calcular e exibir a tabuada desse 
número de 1 a 10.

3) Escreva um programa que simule o funcionamento de um 
cronômetro digital que conta de 00:00 até 01:20. 
Use dois laços para aninhados: o externo para controlar os 
minutos (de 0 a 1) e o interno para controlar os segundos 
(de 0 a 20). Exiba o tempo no formato "Minutos:Segundos".
